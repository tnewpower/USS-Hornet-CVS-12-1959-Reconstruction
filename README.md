# USS-Hornet-CVS-12-1959-Reconstruction
USS Hornet (CVS-12) – 1959 WESTPAC Reconstruction

A historical reconstruction of USS Hornet (CVS-12) as she appeared during her 1959 Western Pacific deployment.

This project combines:

Historical research
Configuration analysis
CAD design
3D printable components
Custom decals
Builder documentation

The objective is to document and preserve the 1959 configuration of USS Hornet through an evidence-based reconstruction suitable for historians, modelers, and the USS Hornet Sea, Air & Space Museum. For modeling purposes, this build is based off the Trumpeter USS Intrepid (CVS-11) 1:350 scale kit.

Project Goals
Document the 1959 configuration using primary photographic evidence.
Produce historically accurate CAD models and STL files.
Publish custom decals for the 1959 air wing.
Record every major configuration decision.
Donate the completed model and research archive to the USS Hornet Museum.
