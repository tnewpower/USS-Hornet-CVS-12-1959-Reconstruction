# USS-Hornet-CVS-12-1959-Reconstruction
This repository contains all the documentation pertaining to my project to reproduce as faithfully as possible the 1959 configuration of the USS Hornet (CVS-12).  STL files are provided for modelers who may wish to reproduce this configuration, using the Trumpeter 1:350 USS Intrepid kit as a starting point. This repository documents the historical reconstruction of
USS Hornet (CVS-12) as configured during her 1959 WESTPAC
deployment.

This is not intended to represent Hornet during the Apollo
recoveries or her later museum configuration.

Every major design decision is supported by primary
photographic evidence whenever possible.
